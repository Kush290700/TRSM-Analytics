## 🚑 Common Failure Modes & Debugging
- **DuckDB Locks:** If queries hang, check for stale lock files in `data/` or `cache/`.
- **Parquet Mismatches:** If `make smoke` fails, the `etl/` likely produced a schema that `app/services/fact_store.py` doesn't expect.
- **RBAC Leaks:** If a user sees data they shouldn't, check `app/core/sensitive_data.py` and `AccessPolicy`.
- **Filter Desync:** If filters don't "stick", check `app/static/js/global_filters.js` and `app/services/filters_service.py`.
